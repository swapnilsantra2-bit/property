# 🏠 Property Registry on Blockchain (Stellar Soroban)

A decentralized **Property Registry DApp** built on the **Stellar Soroban smart contract platform**.
This project enables secure, transparent, and permissionless property registration, ownership transfer, and verification on the blockchain.

---

## 🚀 Live Project Overview

* ⚡ **Fast Transactions:** ~5 seconds finality
* 💰 **Low Cost:** <$0.01 per transaction
* 🌐 **Network:** Stellar Testnet
* 🔗 **Smart Contract Powered by Soroban**
